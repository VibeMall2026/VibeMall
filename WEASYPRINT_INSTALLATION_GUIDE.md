## What is GTK3 Runtime?

GTK3 Runtime provides essential libraries (Cairo, Pango, GObject) that WeasyPrint needs to render HTML to PDF on Windows.

---

## Installation Steps for Windows

### Step 1: Download GTK3 Runtime ⚠️ REQUIRED

**Download Link:** https://github.com/tschoonj/GTK-for-Windows-Runtime-Environment-Installer/releases

1. Go to the releases page
2. Download the latest `.exe` installer:
   - Look for: `gtk3-runtime-3.24.xx-xxxx-xx-xx-ts-win64.exe`
   - Example: `gtk3-runtime-3.24.31-2022-01-04-ts-win64.exe`
3. Run the installer as Administrator
4. Follow the installation wizard (use default settings)
5. **IMPORTANT:** Restart your computer after installation

### Step 2: Verify GTK3 Installation

After restart, check if GTK3 is installed:

```bash
# Check if GTK3 bin directory exists
dir "C:\Program Files\GTK3-Runtime Win64\bin"
```

You should see files like:
- `libgobject-2.0-0.dll`
- `libcairo-2.dll`
- `libpango-1.0-0.dll`

### Step 3: Add GTK3 to System PATH (if needed)

If WeasyPrint still doesn't work after installation:

1. Right-click "This PC" → Properties
2. Click "Advanced system settings"
3. Click "Environment Variables"
4. Under "System variables", find "Path"
5. Click "Edit"
6. Click "New"
7. Add: `C:\Program Files\GTK3-Runtime Win64\bin`
8. Click "OK" on all dialogs
9. Restart your terminal/command prompt

### Step 4: Verify WeasyPrint Works

After GTK3 installation and restart:

```bash
python -c "import weasyprint; print('WeasyPrint version:', weasyprint.__version__)"
```

Expected output:
```
WeasyPrint version: 68.1
```

## Quick Test

Create a test file `test_pdf.py`:

```python
from weasyprint import HTML
from io import BytesIO

html = """
<!DOCTYPE html>
<html>
<head>
    <style>
        body { font-family: Arial; padding: 20px; }
        h1 { color: #FDB913; }
    </style>
</head>
<body>
    <h1>VibeMall Test Invoice</h1>
    <p>This is a test PDF generated by WeasyPrint.</p>
    <p>If you can see this PDF, WeasyPrint is working correctly!</p>
</body>
</html>
"""

try:
    pdf_file = BytesIO()
    HTML(string=html).write_pdf(pdf_file)
    
    with open('test_invoice.pdf', 'wb') as f:
        f.write(pdf_file.getvalue())
    
    print("✅ SUCCESS! PDF generated: test_invoice.pdf")
except Exception as e:
    print(f"❌ ERROR: {e}")
```

Run the test:
```bash
python test_pdf.py
```

## Testing WeasyPrint

Create a test file `test_weasyprint.py`:

```python
from weasyprint import HTML
from io import BytesIO

# Simple HTML
html_content = """
<!DOCTYPE html>
<html>
<head>
    <style>
        body { font-family: Arial; padding: 20px; }
        h1 { color: #FDB913; }
    </style>
</head>
<body>
    <h1>VibeMall Test Invoice</h1>
    <p>This is a test PDF generated by WeasyPrint.</p>
</body>
</html>
"""

# Generate PDF
pdf_file = BytesIO()
HTML(string=html_content).write_pdf(pdf_file)

# Save to file
with open('test_invoice.pdf', 'wb') as f:
    f.write(pdf_file.getvalue())

print("PDF generated successfully: test_invoice.pdf")
```

Run the test:
```bash
python test_weasyprint.py
```

If successful, you'll see `test_invoice.pdf` created in the current directory.
