# WeasyPrint Installation Guide for Windows

## Status: Installation Required

---

## What is WeasyPrint?

WeasyPrint is a Python library that converts HTML/CSS to PDF. It's used in VibeMall to generate invoice PDFs for order confirmation emails.

---

## Installation Steps for Windows

### Step 1: Install GTK3 Runtime (Required)

WeasyPrint requires GTK3 runtime on Windows. Download and install from:

**Download Link:** https://github.com/tschoonj/GTK-for-Windows-Runtime-Environment-Installer/releases

1. Download the latest `.exe` installer (e.g., `gtk3-runtime-3.24.31-2022-01-04-ts-win64.exe`)
2. Run the installer
3. Follow the installation wizard
4. Restart your terminal/command prompt after installation

### Step 2: Install WeasyPrint via pip

Open Command Prompt or PowerShell and run:

```bash
pip install weasyprint
```

### Step 3: Verify Installation

Test if WeasyPrint is installed correctly:

```bash
python -c "import weasyprint; print('WeasyPrint version:', weasyprint.__version__)"
```

If successful, you should see:
```
WeasyPrint version: 60.2 (or similar)
```

## Troubleshooting

### Issue 1: "GTK3 not found" error

**Solution:**
1. Install GTK3 Runtime from the link above
2. Add GTK3 to system PATH:
   - Right-click "This PC" → Properties → Advanced System Settings
   - Click "Environment Variables"
   - Under "System Variables", find "Path"
   - Add: `C:\Program Files\GTK3-Runtime Win64\bin`
   - Restart terminal

### Issue 2: "Cairo not found" error

**Solution:**
- GTK3 Runtime includes Cairo
- Ensure GTK3 is properly installed
- Restart your computer if needed

### Issue 3: "Pango not found" error

**Solution:**
- GTK3 Runtime includes Pango
- Verify GTK3 installation path
- Check system PATH variable

### Issue 4: Import error after installation

**Solution:**
```bash
pip uninstall weasyprint
pip install --no-cache-dir weasyprint
```

## Testing WeasyPrint

Create a test file `test_weasyprint.py`:

```python
from weasyprint import HTML
from io import BytesIO

# Simple HTML
html_content = """
<!DOCTYPE html>
<html>
<head>
    <style>
        body { font-family: Arial; padding: 20px; }
        h1 { color: #FDB913; }
    </style>
</head>
<body>
    <h1>VibeMall Test Invoice</h1>
    <p>This is a test PDF generated by WeasyPrint.</p>
</body>
</html>
"""

# Generate PDF
pdf_file = BytesIO()
HTML(string=html_content).write_pdf(pdf_file)

# Save to file
with open('test_invoice.pdf', 'wb') as f:
    f.write(pdf_file.getvalue())

print("PDF generated successfully: test_invoice.pdf")
```

Run the test:
```bash
python test_weasyprint.py
```

If successful, you'll see `test_invoice.pdf` created in the current directory.
