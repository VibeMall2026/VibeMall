💡 Quick Guide
Promotion area 3 card

Location: First dynamic section of home page (above featured products)

Content: Image only + Link (No title, badge, or description)

Layout: 3 banners side-by-side on desktop (each takes 33% width), stacked on mobile

Use for: Premium promotions, seasonal sales, major product launches
Marketing area 2 card

Location: Below featured products section (replaces hardcoded banners)

Layout: 2 banners side-by-side on desktop, stacked on mobile

Use for: Category highlights, bestsellers, special collections
🖼️ Image Recommendations
  • Recommended Size: 600 x 400 px
  • Aspect Ratio: 3:2 for consistency
  • File Format: JPEG or PNG
  • Display: When 2 banners are present, they split the width 50-50 on desktop, stack on mobile
